WWE 2k19

Also known as World Wrestling Entertainmet 2019 {WWE 2k19}



Minimum System Requirements


  1. OS: Windows 7 (64-bit) (latest updates

  2. Processor: Intel Core i5-3550 / AMD FX 8150

  3. RAM: 4 GB

  4. Hard Drive: 50 GB available space

  5. Video Card: GeForce GTX 670 / Radeon HD 7850

  6. Sound Card: DirectX 9.0c compatible sound card

  7. DirectX: Version 11

  8. Keyboard and Mouse

  9. DVD Rom Drive (or digital purchase)

  10. Internet Connection required for activation

How to Install?



  1. Extract the file using Winrar. (Download Winrar)

  2. Open“WWE 2K19” folder, double click on “Setup” and install it.

  3. Open “CODEX” folder, copy all files and paste it where you install the game

  4. After installation complete, go to the folder where you install the game

  5. Open folder, double click on “WWE 2K19_x64” icon to play the game. Done!
